# Pain Pong

## 概要
Unityで作成した、ブラウザで手軽に遊べるシンプルなポンゲームです。

## プレイ方法
[こちらをクリックしてゲームをプレイ](https://ivix2020.github.io/my-astro-blog/pain-pong/)してください。
キーボードの `↑` キーと `↓` キーでパドルを操作し、ボールを打ち返してください。
